# Images des patchnotes

Déposez ici les captures d'écran référencées dans `src/lib/patchnotes.ts`.

Attendues pour la V1.0 :

- `v1-discover.png` - page Découvrir avec le catalogue TMDB
- `v1-likes.png` - onglet Ma liste
- `v1-dark-mode.png` - application en mode sombre
- `v1-setup.png` - nouvel écran de bienvenue

Format conseillé : PNG, environ 1200px de large (affichées sur ~576px, donc nettes en écran Retina). Une image manquante est simplement masquée dans l'application.
